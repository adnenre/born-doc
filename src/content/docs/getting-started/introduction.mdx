## What is this?

This is a starter template for building a documentation site with Next.js. It's inspired by Nextra, but with a simpler setup and a focus on using modern tools like Tailwind CSS and ShadCN UI.

## Features

- **Next.js App Router**: Uses the latest features of Next.js for the best performance and developer experience.
- **MDX**: Write documentation in Markdown with the power of React components.
- **ShadCN UI**: A beautiful and accessible component library to build your UI.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
- **Lucide Icons**: A clean and consistent icon set.
- **Dark Mode**: Supports light and dark mode out of the box.
- **Search**: Full-text search for your documentation.

## Getting Started

To get started, check out the file `src/app/docs/layout.tsx` to see how the documentation layout is structured. You can add new pages by creating `.mdx` files in the `src/content/docs` directory.
